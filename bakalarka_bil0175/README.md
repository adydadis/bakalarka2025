Aplikace zatim umi pridavat na plochu hradla pomoci tlacitek a propojovat je draty.

Jak se pouziva?
1. Vyberte si hradlo z horniho menu
2. Propojte vystupny a vstupy pomoci dratu (kliknutim na kulicky u hradel)
3. U hradla START lze zmenit hodnotu (0 nebo 1)
4. Kliknete na tlacitko "VYPOCITAT"

Aplikace je nahrana a bezi na mem skolnim webu:
https://homel.vsb.cz/~bil0175/
